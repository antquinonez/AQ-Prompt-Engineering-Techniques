## Terms
- __Ask__: The input to a model; I also call this a 'Prompt'.
- __Context Overfitting__:  During a sessions, when a model fails to work well because it's locked into the wrong prompt content or responses.
- __Machine Learning__:  The process of training a model to perform a task.
- __Output__:  The output/answer of a model; for example, what a model says in response to a prompt.
- __Prompt__: The input to a model; I also call this an 'Ask'.
- __Prompt Engineering__:  The process of creating a prompt that will generate the desired output.
- __Prompt Overfitting__: When a model fails to work well because it's locked into the wrong prompt content.
- __Response Overfitting__: When a model fails to work well because it's locked into the wrong prompt responses.
- __Training__: The process of training a model to perform a task.
